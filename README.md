### Prerequisites
- Node.js (v16+ recommended)
- npm or Yarn

### Installation
1. Clone the repository:
```bash
   git clone https://github.com/PavilionRYZ/Subhra.dev.git
   ```
2. Install dependencies:
 ```bash
npm install
```
3. Run locally:
```bash
npm run dev
```
   Open http://localhost:5173 in your browser.


4. Build for production:
```bash
npm run build
```
